# Python Library Specialist - Quick Reference

## Purpose

Maintains the public API surface, Pydantic models, type safety, and packaging for a pip-installable Python library using src layout, hatchling, ruff, and mypy strict.

## When to Use

- Implementing or extending the library's public API
- Creating Pydantic models or data classes
- Managing package structure and imports
- Ensuring type safety and PEP 561 compliance
